Qu'est-ce que le Fleiss Kappa ?

Le Fleiss Kappa mesure l'accord entre plusieurs annotateurs. Plus la valeur est proche de 1, plus l'accord est fort.

  • κ = 1 : Accord parfait
  • κ = 0 : Accord aléatoire
  • κ < 0 : Désaccord systématique
